## 遇到的问题
1. **Electron 无法初始化**：在 bash 终端环境中，Electron 的 GUI 进程初始化失败，`require('electron')` 返回路径字符串而非 API。调整策略为先用 Node.js 服务器跑通核心功能，Electron 包装放到后续用脚手架重试。
2. **npm 网络不稳定**：使用中国镜像 (`npmmirror.com`) 加速 Electron 二进制下载。
3. **Windows 文件锁**：node_modules 删除时遇到 EBUSY 错误，需先 kill node 进程。

## 遇到的问题
3. **Electron 在 VS Code bash 终端无法启动 GUI**：这是环境限制（MSYS2/Git Bash 无 Windows GUI 子系统访问权）。创建了 start-electron.bat 供用户在 Windows 资源管理器中双击启动。实际打包为 .exe 后不存在此问题。
4. **init() 函数不是 async**：之前 server/index.js 自执行调用使用了 `.catch()`，但 init() 是同步函数。改为 try/catch 包裹。

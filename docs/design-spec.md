# UI 设计规范 — ClipSync

## 1. 设计原则

- **简洁**：去除一切非必要元素，一目了然
- **直观**：操作符合直觉，无需学习成本
- **一致**：三端 UI 风格统一
- **轻量**：快速响应，不卡顿

## 2. 颜色系统

| 用途 | 变量名 | 色值 | 预览 |
|------|--------|------|------|
| 主背景 | `--bg-primary` | `#F0F7FF` | 淡蓝白 |
| 卡片背景 | `--bg-card` | `#FFFFFF` | 纯白 |
| 顶栏背景 | `--header-bg` | `#87CEEB` | 天蓝 |
| 主按钮 | `--btn-primary` | `#4A90D9` | 蓝色 |
| 按钮悬停 | `--btn-hover` | `#357ABD` | 深蓝 |
| 危险按钮 | `--btn-danger` | `#E74C3C` | 红色 |
| 文字主色 | `--text-primary` | `#333333` | 深灰 |
| 文字辅助 | `--text-secondary` | `#999999` | 浅灰 |
| 边框 | `--border` | `#E8E8E8` | 浅灰边框 |
| 在线状态 | `--online` | `#52C41A` | 绿色 |
| 警告/即将过期 | `--warning` | `#FA8C16` | 橙色 |
| 选中/高亮 | `--active` | `#E3F2FD` | 浅蓝高亮 |

## 3. 字体

| 平台 | 字体栈 |
|------|--------|
| Windows | `"Microsoft YaHei", "Segoe UI", sans-serif` |
| iPadOS | `-apple-system, "PingFang SC", "Helvetica Neue", sans-serif` |
| HarmonyOS | `"HarmonyOS Sans", "PingFang SC", sans-serif` |

- 主文字大小：14px
- 标题大小：18px
- 辅助文字：12px

## 4. 间距系统

| 级别 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 紧凑间距 |

| md | 16px | 卡片内边距、列表间距 |
| lg | 24px | 区块间距 |

## 6. 内容卡片设计

### 6.1 文字卡片
```
┌──────────────────────────────────────┐
│ 📝 文字内容预览（最多显示 3 行）...    │
│                                      │
│ 2 分钟前 · 来自 电脑          [复制]  │
└──────────────────────────────────────┘
```
- 高度：自适应（最小 72px）
- 点击展开全文
- 「复制」按钮复制到剪贴板

### 6.2 图片卡片
```
┌──────────────────────────────────────┐
│ [缩略图]    screenshot.png            │
│ [150×100]  1.2 MB                    │
│                                      │
│ 1 小时前 · 来自 电脑    [查看] [保存] │
└──────────────────────────────────────┘
```
- 左侧缩略图（150×100px）
- 「查看」打开大图预览
- 「保存」下载到本地

### 6.3 文件卡片
```
┌──────────────────────────────────────┐
│ 📄 项目文档.docx                      │
│    856 KB                            │
│                                      │
│ 3 小时前 · 来自 手机    [下载] [打开] │
└──────────────────────────────────────┘
```
- 根据扩展名显示不同图标
- 「下载」保存到本地
- 「打开」用系统默认程序打开

## 7. 输入区设计

```
┌──────────────────────────────────────┐
│  [📎 选择文件]                        │
│  ┌────────────────────────────────┐  │
│  │ 输入文字或粘贴内容...           │  │  ← 自动伸缩高度
│  │                                │  │
│  └────────────────────────────────┘  │
│                           [发送 →]   │
└──────────────────────────────────────┘
```

- 支持 Ctrl+V 粘贴图片和文字
- 支持拖拽文件到输入区
- 发送按钮在主按钮色

## 9. 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 桌面 | ≥ 768px | 最大宽度 800px，居中 |
| 移动端 | < 768px | 全宽，边距 16px |

## 10. 交互动画

- 卡片出现：淡入 + 上滑（300ms ease-out）
- 卡片删除：淡出 + 左滑（200ms ease-in）
- 新内容通知：顶栏高亮闪烁
- 按钮点击：轻微缩放（scale 0.97）
- 加载中：淡蓝色旋转图标

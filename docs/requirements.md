# 需求规格说明 — ClipSync

## 项目概述

ClipSync 是一个局域网跨设备剪贴板/内容同步工具，允许同一 WiFi 下的 Windows 电脑、iPadOS 平板、HarmonyOS 手机之间实时同步文字、图片和文件。

## 核心需求

### R1 — 跨平台支持
- Windows 电脑：Electron 桌面应用（exe 安装包）
- iPadOS 平板：PWA（Safari 浏览器访问 + 添加到主屏幕）
- HarmonyOS 手机：PWA（浏览器访问 + 添加到桌面）

### R2 — 内容同步
- 支持文字、图片、文件三种类型
- 任意一端发送内容后，其他在线设备实时接收
- 手机/平板同样可以发送内容给电脑和其他设备

### R3 — 24 小时生命周期
- 所有同步内容保存 24 小时后自动删除
- 过期内容包括数据库记录和关联的磁盘文件
- 每 60 秒定期检查 + 启动时立即清理
- 新加入设备只同步未过期内容

### R4 — 局域网通信
- 设备必须在同一 WiFi 网络下
- 通过 mDNS 自动发现，无需手动输入 IP
- 使用 HTTP + WebSocket 协议通信

### R5 — 设备安全授权
- 新设备首次连接需电脑端审批
- 批准后可选择「记住此设备」，后续自动连接
- 电脑端可查看和管理已授权设备列表
- 可随时移除任何设备的授权

### R6 — UI 设计
- 简洁直观的界面风格
- 淡蓝色主色调
- 内容以卡片形式展示
- 支持按类型筛选（全部/文字/图片/文件）

### R7 — 去中心化
- 没有固定主节点，任何可运行服务器的设备都可担任
- 多个服务器节点时可冗余连接
- PC 不在线时，移动端可离线查看本地缓存

## 非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 局域网内文字同步延迟 < 1 秒 |
| 可靠性 | 断线自动重连 |
| 可用性 | 24h 过期机制确保存储空间可控 |
| 安全性 | 设备白名单 + Token 认证 |
| 可维护性 | 代码结构清晰，文档完备 |
